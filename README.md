# укр

Встановлювач авторських програм.

## Встановлення

```
curl -o- https://raw.githubusercontent.com/kohutd/ukr/main/install.bash | bash
```

## Використання

### Встановлення версії програми

```shell
укр встановити <програма> <версія>
```

### Встановлення останньої версії програми

```shell
укр встановити <програма>
```

### Видалення версії програми

```shell
укр видалити <програма> <версія>
```

### Видалення всіх версій програми

```shell
укр видалити <програма>
```

### Зміна поточної версії програми

```shell
укр використовувати <програма> <версія>
```

### Отримання поточної версії програми

```shell
укр використовується <програма>
```

### Отримання встановлених програм або версій програми

```shell
укр встановлені [програма]
```

### Отримання доступних програм або версій програми

```shell
укр доступні [програма]
```
